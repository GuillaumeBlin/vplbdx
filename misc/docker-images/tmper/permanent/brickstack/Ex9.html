Consignes

  • ⚠ Appuyez sur le bouton de soumission d'un nouveau devoir (i.e. Ajouter un travail) pour commencer l'exercice
  • Après chaque modification de la pile, cliquez sur Add to current Gif pour valider le nouvel état de la pile de votre réponse
  • Pour recommencer l'exercice, cliquez sur New Gif (votre réponse actuelle sera perdue)
  • Vous devez respecter la convention de passage de paramètres de l'architecture x86 vue en cours intégré
  • Les valeurs doivent être toutes stockées en hexadécimal 
    (décimal) → (hexa)
    (char) → (ascii)
  • Pour rendre le devoir, vous devez cliquer sur Save the Gif, le stocker sur votre ordinateur et le déposer dans la zone prévue à cet effet (i.e. Remises de fichiers)
